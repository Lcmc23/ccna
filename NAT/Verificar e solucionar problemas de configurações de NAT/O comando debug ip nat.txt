O comando debug ip nat

    * (Asterisco) - O asterisco ao lado do NAT indica que a conversão está ocorrendo no caminho fast-switched. O primeiro pacote de uma 
    conversa é sempre process-switched, o que é mais lento. Os outros pacotes passam pelo caminho fast-switched se houver uma entrada na cache.
    
    s= - Este símbolo se refere ao endereço IPv4 de origem.
    
    a.b.c.d--->w.x.y.z - Esse valor indica que o endereço origem a.b.c.d é convertido a w.x.y.z.
    
    d= - Este símbolo se refere ao endereço IPv4 de destino.
    
    [xxxx] - O valor entre parêntesis é o número de identificação do IPv4. Essas informações podem ser úteis para depuração, pois permitem 
    correlação com outros rastreamentos de pacotes de analisadores de protocolos.
